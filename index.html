HTML - Hypertext Markup Language

HTML and CSS are the two languages used to create web pages. HTML provides the content of the web pages whereas CSS beautifies the content and all the space around it.
More information can be found here and here.

HTML Elements
The objects, or "words", used to create a web page are elements.
Elements consist of three things:
  • Opening tag - < name >*
  • Content - Visible text
  • Closing tag - < /name >

    • For example, writing "<b> bold </b>" makes the word bold.

    *How to use special characters


    HTML Attributes
    To provide a link or an image on a webpage, attributes are used.

    A link in a webpage can be seen as an actionable element - the address to another webpage is embedded into the visible text, allowing users to access other webpages easily with just a click. A link is written as follows:
    "< a href="address to the other webpage" > visible text < /a >"
    where**
    a stands for link element and href means attribute name.

    A image in a webpage can be seen as the visible representation of another webpage - the address to another webpage that houses only a picture is activated to show a picture. An image is shown as follows:
    "< img src="address to the picture webpage" alt = "error text" >
    where img stands for image element, src means source, and alt provides a means to show an error message if the image address fails.



    The image element differs from the link element as it doesn't have a closing tag - this is called having a void tag.
